EU Digital COVID Certificate

Import and store easily your EU COVID Certificates on your smartphone.

See your certificates Import your first certificate

This website is a web application that allows you to manage and show your Digital EU COVID certificates (DCC), using your device's camera to import them.

This application can even be installed on your homepage, so you can use it even while offline.

The certificates are safely stored locally on your device. No contact with any online service is made during use.

An EU Digital COVID Certificate is a digital proof that a person has either:

  •   been vaccinated against COVID-19
  •   received a negative test result or
  •   recovered from COVID-19

Key features of the certificates are:

  •   Digital and/or paper format
  •   with QR code
  •   free of charge
  •   in national language and English
  •   safe and secure
  •   valid in all EU countries

More information is available on the European Commission website .

This application was developed because in some countries, the official application is either bloated with other features, asking far too many permissions to run, or even not available outside Android and iOS.

As COVID-19 is still looming around, and its variants are getting stronger and stronger, it seems inevitable that governments will rely more and more on these certificates to avoid setting up lockdowns again.

This application is meant to be usable on any browser and device, and offer to anyone a simple and quick way to manage their certificates. Nothing more.

Open source

Because health information is sacred and private, this application has its source code available for anyone to see and audit. As a simple web application without any server component, it can be self-hosted for maximum privacy.

The source code is licensed under the MIT license, and is accessible on this Github repository. The Readme file will guide you if you want to download or compile the application yourself.

If you encounter a bug, feel free to file a new issue. All contributions are welcome, and most especially translations to other languages.
